# Kalash Harchandani | Premium Portfolio

A high-fidelity, tech-themed developer portfolio built with React, Tailwind CSS, and Framer Motion. 

## 🚀 Experience the Immersive UI
- **Pure Black Aesthetic**: Designed with a true-black backdrop for a sleek, mission-critical look.
- **Morphing Navbar**: A floating, curvy navigation bar that adapts dynamically as you scroll.
- **Massive Whitespace**: High-impact Hero section with "breathing space" design architecture.
- **Tech Stack Logos**: Integration of branded corporate logos for every technology in the arsenal.
- **Responsive & Interactive**: Fully optimized for mobile with smooth, stagger-animated section transitions.

## 🛠️ Built With
- **Frontend**: React (Create React App), Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React & React Icons
- **Design Strategy**: Minimalist Glassmorphism & Cyber-Tech aesthetic

## 📂 Features
- **Project Case Studies**: Massive detail cards with overlapping layouts.
- **Professional Path**: Interactive timeline featuring industry legacy.
- **Direct Resume Access**: One-tap PDF download directly from the Hero section.
- **AI-Driven Tagline**: Real-time typewriter effect highlighting core competencies.

## 🏁 Getting Started
1. Clone the repository
2. Install dependencies: `npm install`
3. Run locally: `npm start`
4. Build for production: `npm run build`
